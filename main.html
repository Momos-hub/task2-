सङ्ख्या




Numbers can be used as adjective ( विशेषणम् ) or adverbs ( क्रियाविशेषणम् )


सङ्ख्या

  1. विशेषणम्

    1. सङ्ख्यावाचकम्
      ( कति )

      1. This indicates the count

        ( how many )

    2. कमवाचकम् ( कतम् )

      1. This indicates the position rank or order

        ( which one )

  2. क्रियाविशेषणम्

    2. आवृत्तिवाचकम् ( कतिवारम् )

      1. This indicates how many times an action is performed.
